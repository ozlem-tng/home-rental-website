# 🏠 Home Rental Website

A modern and responsive **frontend rental listing website** designed to showcase properties for sale or rent. This project focuses on UI/UX design and layout using HTML, CSS, and Bootstrap.

---

## 🚀 Features

* 🏡 Property listing cards (sale & featured)
* 🔍 Search bar for property exploration
* 📂 Multi-page structure (home, listing, category, detail pages)
* 📱 Fully responsive design (mobile-friendly)
* 🎞️ Slider/Carousel using Swiper.js
* ⚡ Lazy loading images for performance optimization

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Swiper.js
* LazySizes
